## 1. アプリ概要

### 1.1 基本情報
| 項目 | 内容 |
|------|------|
| **アプリ名** | ハザクラ |
| **読み方** | はざくら / Hazakura |
| **タグライン** | 「創作の未来をともに描くパートナー」 |
| **カテゴリ** | AI創作支援アプリ / ライティングエディター |
| **ターゲット** | 小説家、作家、Web小説作家、脚本家、同人作家 |

### 1.2 名前の由来・意味
「ハザクラ（葉桜）」は、桜の花が散った後に若葉が芽吹く時期の桜を指す日本語です。

**象徴するもの：**
- 🌸 **終わりと始まり** - 花が散っても、新たな命（葉）が芽吹く
- 🍃 **成長と変化** - 創作活動における成長のプロセス
- 🇯🇵 **日本の美意識** - 儚さと希望が共存する美しさ
- ✍️ **創作の循環** - 一つの作品が終わり、次の創作が始まる

## 2. アプリの主要機能

### 2.1 コア機能
1. **AIパートナー** - 2つのAI（Grok + Gemini）が執筆をサポート
   - Grok: メイン執筆支援AI（続き生成、キャラ深掘り、プロット相談）
   - Gemini: 分析・評価AI（感情曲線、作品成分分析）

2. **執筆エディタ** - プロ仕様のライティング環境
   - ZENモード（集中執筆）
   - 縦書き/横書き切り替え
   - 音声読み上げ機能

3. **資料集** - 創作に必要な情報を一元管理
   - ジャンル、キャラクター、世界観、専門用語

4. **プロット構成** - 物語の骨組みを整理
   - 起承転結、三幕構成などのテンプレート

5. **作品分析** - AIが作品を客観的に評価
   - 感情曲線の可視化
   - ジャンル成分のレーダーチャート

6. **シリーズ管理** - 長編・連載作品の管理
   - 親子構造でのエピソード管理
   - 設定の共有機能

## 4. ロゴ要件

### 4.1 必須要件
| 項目 | 要件 |
|------|------|
| **形状** | 正方形ベース（アプリアイコン対応） |
| **サイズ展開** | 16x16px（ファビコン）〜 1024x1024px（フル解像度） |
| **形式** | PNG（透過）、SVG推奨 |
| **背景** | 透過 or カラー背景（両方あると望ましい） |

### 4.2 使用箇所
1. **ファビコン** - ブラウザタブのアイコン（16x16, 32x32px）
2. **アプリロゴ** - ナビゲーションバー（40x40px程度）
3. **ログイン画面** - 大きめのロゴ表示（80x80px程度）
4. **OGP画像** - SNSシェア用（1200x630px内に配置）
5. **スプラッシュ** - 起動画面（将来的なPWA対応）

### 4.3 デザインの方向性

#### 推奨スタイル
- ✅ **モダン・ミニマル** - シンプルで洗練されたデザイン
- ✅ **日本的美意識** - 和のエッセンスを取り入れる
- ✅ **クリエイティブ** - 創作・執筆を連想させる
- ✅ **温かみ** - AIだけど人間味のある印象

#### 避けるべきスタイル
- ❌ 過度に複雑なデザイン（小さいサイズで視認性が落ちる）
- ❌ 写実的なイラスト（アイコンとして機能しにくい）
- ❌ 冷たい・機械的な印象

## 5. デザインインスピレーション

### 5.1 モチーフ候補

#### A. 葉桜モチーフ
- 桜の花びら + 若葉の組み合わせ
- 花から葉への変化を表現
- ピンク → グリーンのグラデーション

#### B. 執筆・創作モチーフ
- 羽ペン（クイル）
- 開いた本
- インクのしずく
- 巻物（スクロール）

#### C. 文字・タイポグラフィ
- 「葉」または「桜」の漢字をモダンにアレンジ
- ひらがな「は」をシンボル化
- 「H」をロゴマーク化

#### D. 複合モチーフ（推奨）
- **葉桜 + 羽ペン** - 創作と季節の融合
- **桜の花びら + ペン先** - 執筆の象徴
- **本から芽吹く葉** - 物語から生まれる創造

### 5.2 参考イメージワード
```
和モダン、桜、葉桜、春、新緑、成長、
執筆、創作、物語、インスピレーション、
AIパートナー、寄り添う、温かい、知的
```

## 6. 技術的要件

### 6.1 ファイル形式
| 形式 | 用途 | 必須/任意 |
|------|------|-----------|
| **PNG** | Webファビコン、アプリ内表示 | 必須 |
| **SVG** | 拡大縮小に強い、CSS制御可能 | 強く推奨 |
| **ICO** | Windows用ファビコン | 任意 |

### 6.2 サイズバリエーション
```
favicon-16x16.png   - ブラウザタブ
favicon-32x32.png   - 高解像度タブ
favicon-180x180.png - Apple Touch Icon
favicon-512x512.png - PWA用
logo-1024x1024.png  - フル解像度
```

### 6.3 カラーバリエーション
- **フルカラー版** - 通常使用
- **モノクロ版** - 印刷物、シンプルな背景用
- **白抜き版** - ダークモード、暗い背景用
- **アイコンのみ版** - テキストなし、小サイズ用
